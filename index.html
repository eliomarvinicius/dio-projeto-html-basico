História da web

(Voltar)

A Web surgiu em 1989, criada por Tim Berners-Lee, no CERN. No começo, era só uma forma de compartilhar documentos científicos, mas logo se expandiu. Nos anos 90, surgiram navegadores como o Netscape, e a internet começou a bombar. Com o tempo, vieram novas tecnologias, como CSS e JavaScript, deixando os sites mais bonitos e interativos. Hoje, a Web está em tudo: redes sociais, lojas online, serviços de streaming.

Relação cliente X Servidor

(Voltar)

Imagina que você quer assistir a um vídeo no YouTube. Seu navegador (cliente) pede esse vídeo para o YouTube (servidor), que envia o conteúdo de volta. Esse é o modelo cliente-servidor: um lado pede, o outro responde. Essa comunicação acontece através de protocolos como HTTP. Além disso, existem servidores para armazenar bancos de dados, rodar aplicativos e até gerenciar mensagens em tempo real. Basicamente, toda a internet funciona com essa troca de informações!

Linguagens de Programação

(Voltar)

Linguagens de programação são o que os programadores usam para falar com os computadores. Elas podem ser mais simples ou mais complexas, dependendo da necessidade. Algumas são mais próximas da máquina, como Assembly, enquanto outras são mais fáceis de entender, como Python e JavaScript. Também tem linguagens que precisam ser traduzidas antes de rodar (C, Java) e outras que são executadas direto (Python, JavaScript). Cada uma tem seu uso: algumas são boas para sites, outras para games, inteligência artificial e até sistemas bancários

HTML: A Base da Web

(Voltar)

HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas na web. Ele funciona como o esqueleto de um site, organizando os conteúdos e dizendo ao navegador o que deve ser exibido e como os elementos estão dispostos. Diferente de uma linguagem de programação, HTML não executa lógicas ou cálculos. Ele apenas marca e organiza o conteúdo. Isso é feito por meio de tags, que são comandos que indicam qual tipo de informação está sendo inserida.

Tags HTML

(Voltar)

HTML é a base dos sites. Ele usa "tags" para organizar o conteúdo da página. Por exemplo: <h1> até <h6> são títulos, <p> é parágrafo, <strong> deixa em negrito.
<img> adiciona imagens, <a> cria links, <button> faz botões.
<table> serve para tabelas e <ul> e <ol> para listas.
<u> faz um sublinhado na palavra marcada como no exemplo cachorro.
<i> deixa o texto em um efeito itálico
<mark> faz com que a palavra fique com um efeito de marcado
<strong> deixa o texto em negrito
<sup> deixa um texto sobrescrito como em: H2O.
<sub> deixa o texto subscrito como em : CO2.
<blockquote> é usada para destacar citações longas, geralmente exibindo um recuo para indicar que se trata de uma citação como em:

"A simplicidade é o último grau da sofisticação." - Leonardo da Vinci

<del> É usada para indicar que um texto foi removido ou riscado, geralmente para mostrar edições ou mudanças em um documento. Ex: O preço original era R$ 100,00, agora está por R$ 80,00!
<abbr> É usada para indicar abreviações ou siglas. Quando o usuário passa o mouse sobre a sigla, um texto explicativo aparece. Ex: HTML

O HTML sozinho cria a estrutura do site, mas para deixá-lo bonito, usamos CSS, e para dar vida com interatividade, entra o JavaScript.
